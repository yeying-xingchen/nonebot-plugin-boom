## 💿 安装

<details open>
<summary>使用 nb-cli 安装</summary>
在 nonebot2 项目的根目录下打开命令行, 输入以下指令即可安装

    nb plugin install nonebot-plugin-boom

</details>

<details>
<summary>使用包管理器安装</summary>
在 nonebot2 项目的插件目录下, 打开命令行, 根据你使用的包管理器, 输入相应的安装命令

<details>
<summary>pip</summary>

    pip install nonebot-plugin-boom
</details>
<details>
<summary>pdm</summary>

    pdm add nonebot-plugin-boom
</details>
<details>
<summary>poetry</summary>

    poetry add nonebot-plugin-boom
</details>
<details>
<summary>conda</summary>

    conda install nonebot-plugin-boom
</details>

打开 nonebot2 项目根目录下的 `pyproject.toml` 文件, 在 `[tool.nonebot]` 部分追加写入

    plugins = ["nonebot_plugin_boom"]

</details>
